# Sales OS — Fluxo do Cliente

Ciclo completo: **fechamento → entrega → acompanhamento recorrente**.

Princípio: processo sólido primeiro, IA depois. Nada de automação antes do fluxo comercial estar claro.

---

## Visão geral

| Fase | Arquivo | Duração típica | Entregável-chave |
|------|---------|---------------|-----------------|
| 0 — Pré-venda | [[fase-0-prevenda]] | 2–4 semanas | Contrato assinado + setup pago |
| 1 — Kickoff & Diagnóstico | [[fase-1-kickoff-diagnostico]] | 1 semana | Relatório de diagnóstico |
| 2 — Estrutura de Processo | [[fase-2-estrutura-processo]] | 2–3 semanas | Fluxograma + Playbook |
| 3 — Gate | [[fase-3-gate]] | 2–5 dias | Go/No-Go + pacote escolhido |
| 4 — Execução | [[fase-4-execucao]] | 1–4 semanas | Sistema em produção + time treinado |
| 5 — Acompanhamento | [[fase-5-acompanhamento]] | contínuo | Renovação ou case |

**Total kickoff → go-live:** 5–8 semanas (rota 1) ou 6–9 semanas (rota 2).
