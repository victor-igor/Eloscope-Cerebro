## ✅ O que você precisa preparar

### 1. WhatsApp Business ativo

O Sales OS conecta ao WhatsApp dos seus vendedores para a IA Copiloto funcionar.

**Cada vendedor que vai usar precisa:**
- Ter o **WhatsApp Business** instalado no celular (não o WhatsApp comum)
- Usar um número de celular dedicado ao trabalho (não o número pessoal misturado com pessoal)
- Ter o número verificado e ativo (consegue enviar e receber mensagens normalmente)

**Para baixar o WhatsApp Business:**
- Android: Play Store → buscar "WhatsApp Business"
- iPhone: App Store → buscar "WhatsApp Business"

> ⚠️ **Importante:** o número que o vendedor usa hoje para falar com clientes é o que conectamos. Não precisa trocar de número — só garantir que é o Business.

---

### 2. Acesso ao WhatsApp Web / QR Code

Para conectar, Hugo vai precisar que o vendedor escaneie um QR Code (como faz quando usa WhatsApp Web no computador).

**O que vai acontecer:**
1. Hugo envia um QR Code (imagem ou link)
2. O vendedor abre o WhatsApp Business no celular
3. Vai em: Menu → Aparelhos conectados → Conectar um aparelho
4. Escaneia o QR Code
5. Pronto — conectado

**Isso leva menos de 2 minutos por vendedor.**

> Não precisa instalar nada no computador. O celular continua funcionando normalmente.

---

### 3. Conta de IA (LLMs) — custo direto seu

O Sales OS usa modelos de inteligência artificial (como o ChatGPT / Claude) para processar as conversas e gerar os scripts personalizados. **O custo desses modelos é por sua conta** — mas é baixo e você contrata diretamente.

**Recomendação:** OpenAI (ChatGPT API)

**Como criar sua conta:**
1. Acesse: platform.openai.com
2. Crie uma conta com seu e-mail
3. Vá em "Billing" e adicione um cartão de crédito
4. Coloque um limite de gasto mensal (recomendamos R$200 para começar)
5. Gere uma "API Key" (chave de acesso) e envie para Hugo

**Custo estimado:** R$50–200/mês dependendo do volume de conversas do time.

> Se preferir usar o Claude (Anthropic) ou outro modelo, Hugo pode configurar — só avisar.

## ❓ Dúvidas frequentes

**"O WhatsApp vai parar de funcionar no meu celular?"**
Não. O celular continua funcionando normalmente. A conexão é como o WhatsApp Web — você pode usar o celular e o sistema ao mesmo tempo.

**"O cliente vai saber que tem uma IA na conversa?"**
Não. A IA Copiloto funciona como um assistente invisível para o vendedor — ela aparece para o vendedor (sugestões, scripts, alertas), não para o cliente. Quem escreve é sempre o vendedor.

**"E se o celular ficar sem bateria ou internet?"**
A IA Copiloto precisa do celular conectado para funcionar. Recomendamos que os vendedores mantenham o celular carregado e com internet durante o horário de trabalho.

**"Posso conectar um número já usado por um vendedor há anos?"**
Sim. O número não muda, os contatos não somem. A conexão é adicional — não altera nada do que já existe.
