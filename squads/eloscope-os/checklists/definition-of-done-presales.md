# Definition of Done — Presales

Aplicável aos agents: orion, sirius, andromeda, halley.

## Antes de passar lead pra discovery (orion)
- [ ] ICP score ≥ 50 (Polaris ou equivalente)
- [ ] BANT básico preenchido (B, A, N, T)
- [ ] Dossier do Sírius anexado
- [ ] Próxima ação recomendada documentada
- [ ] Handoff criado em `leads/{slug}/qualification.md`

## Antes de enviar proposta (andromeda)
- [ ] Discovery report completo lido
- [ ] VPC preenchido (Prisma)
- [ ] 2-3 cases similares selecionados (Ecos)
- [ ] Pricing tier justificado (Mercúrio)
- [ ] OUT of scope explícito
- [ ] Objeções antecipadas mapeadas (Égide pré-ativado)
- [ ] Handoff Andrômeda → Íris criado (quando Íris existir; hoje gera deck direto)
- [ ] Proposta revisada internamente antes de enviar

## Antes de parar follow-up (halley)
- [ ] 4 touches realizados (dia 2, 5, 10, 20)
- [ ] Cada touch variou ângulo (confirmação → valor → decisão → breakup)
- [ ] Stage atualizado no HubSpot
- [ ] Se won: handoff pra Regente com contrato
- [ ] Se lost: motivo documentado (pricing | timing | authority | concorrência | outro)

## Sempre
- [ ] Artefato estruturado em cada handoff (nunca chat livre)
- [ ] Nomenclatura de arquivos correta (`handoff-{from}-{to}-{date}.md`)
- [ ] Sem PII em ambiente público (Pallas valida)
