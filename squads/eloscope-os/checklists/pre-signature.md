# Checklist Pré-Assinatura

> Executar antes de enviar qualquer contrato para assinatura.
> Todos os itens bloqueantes devem estar ✅ para GO.

## Estrutura do contrato [BLOQUEANTE]

- [ ] Partes completas (nome, CNPJ/CPF, endereço, representante legal)
- [ ] Objeto claro e específico (não genérico)
- [ ] Entregáveis numerados e descritos
- [ ] Exclusões explícitas (o que NÃO está incluído)
- [ ] Prazo com datas absolutas (DD/MM/YYYY, não "após X dias")
- [ ] Valor total e forma de pagamento definidos
- [ ] Cláusula de rescisão com aviso prévio

## Compliance [BLOQUEANTE]

- [ ] Nenhuma cláusula ilegal identificada (Sólon)
- [ ] Nenhuma cláusula abusiva (CDC art. 51 se B2C)
- [ ] Confidencialidade presente com prazo definido
- [ ] Propriedade intelectual clara

## Pagamento [BLOQUEANTE]

- [ ] Multa por atraso definida (máx 2%)
- [ ] Juros de mora definidos (máx 1% a.m.)
- [ ] Cláusula resolutiva expressa (suspensão por inadimplência)
- [ ] Dados bancários da contratada no contrato ou em anexo

## Formalidades [BLOQUEANTE]

- [ ] Foro definido (cidade e estado)
- [ ] Data do documento preenchida
- [ ] Espaços para assinatura das partes

## Recomendado [NÃO BLOQUEANTE]

- [ ] Testemunhas previstas (título executivo extrajudicial)
- [ ] Referência à proposta comercial que originou o contrato
- [ ] Correção monetária definida (IPCA ou IGPM)
- [ ] Seguro/garantia adicional (se ticket > R$10k)
- [ ] Revisão por advogado (se ticket > R$20k ou cliente exigir)

## Veredicto

| Resultado | Condição |
|-----------|----------|
| **GO** | Todos os itens BLOQUEANTES ✅ |
| **NO-GO** | Qualquer item BLOQUEANTE ❌ — listar pendências |
