# Definition of Done — Delivery

Aplicável aos agents: mentor, temis, clio.

## QA Gate (temis)
- [ ] Jornadas críticas testadas end-to-end
- [ ] Aceite do contrato validado item por item
- [ ] Performance: CWV todos verdes (Cometa)
- [ ] Acessibilidade: WCAG AA mínimo
- [ ] Responsivo em mobile/tablet/desktop
- [ ] Cross-browser: Chrome, Safari, Firefox, Edge
- [ ] Segurança: OWASP top 10 revisto (Argos)
- [ ] Sem PII real em staging
- [ ] Rollback plan documentado e testado
- [ ] Zero 🔴 crítico aberto

## Documentação (clio)
- [ ] User manual em linguagem do usuário
- [ ] Runbook com operação + troubleshooting + escalation matrix
- [ ] FAQ organizado por dor, não por feature
- [ ] Video scripts (3-5min cada) das jornadas principais
- [ ] Onboarding deck pra primeira semana
- [ ] Screenshots atualizados
- [ ] Seções LGPD: exportar dados + deletar dados

## Handover (mentor)
- [ ] Cronograma de treinamento combinado
- [ ] Treinamento de usuários finais realizado
- [ ] Treinamento de admins realizado
- [ ] Go-live checklist 100%
- [ ] Dados migrados e validados
- [ ] DNS/domínio apontado
- [ ] Monitoring ativo
- [ ] Sign-off formal do cliente obtido
- [ ] Handoff pra Elo (CS) com baseline + aha moment + milestones 30/60/90

## Sempre
- [ ] Artefato estruturado em cada handoff
- [ ] Sem exceções: zero 🔴 = pré-requisito de GO
- [ ] Contrato vira lei — não entregamos menos, não prometemos mais
