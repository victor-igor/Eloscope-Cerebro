## Tom de Voz

| Eixo | Posição | Score |
|------|---------|-------|
| Formal ←→ Casual | Ligeiramente casual | 6/10 |
| Sério ←→ Lúdico | Sério com humanidade | 3/10 |
| Respeitoso ←→ Irreverente | Respeitoso e direto | 3/10 |
| Entusiasmado ←→ Factual | Entusiasmado com base real | 6/10 |

**Persona da marca:** O CTO que largou a grande empresa para montar o próprio estúdio. Resolve problemas reais enquanto outros ainda estão no PowerPoint.

### ✅ Faça
- Use "você" e "seu negócio"
- Seja específico com prazo/número ("Em 3–10 dias úteis", não "rapidamente")
- Mostre o resultado antes do processo
- Use verbos de ação: automatize, implemente, escale, entregue

### ❌ Evite
- "Nossa empresa oferece soluções" · "Revolucionário" · "disruptivo"
- "Ecossistema" · "holístico" · "plataforma completa"
- Explicar tecnologia antes do benefício

### Tom por Canal
| Canal | Ajuste |
|-------|--------|
| LinkedIn (post) | Direto, provocativo, termina com insight ou pergunta |
| LinkedIn (bio) | Ultra-objetivo, resultado em destaque |
| WhatsApp comercial | Casual mas profissional, vai ao ponto |
| Proposta comercial | Confiante, estruturado, sem floreios |
| E-mail frio | 3 parágrafos: problema → solução → CTA |
| Site (hero) | Impacto máximo em 10 palavras |

## Vocabulário

**Sempre usamos:** agentes · squads · automação · operação · implementação · resultado · transformação · escalar · entregar · produção

**Evitamos:**
| ❌ | ✅ |
|----|-----|
| disruptivo | diferente, à frente |
| revolucionário | novo, inédito no mercado |
| soluções | agentes, implementação |
| plataforma | estúdio, squad, agente |
| chatbot | agente (implica raciocínio, autonomia) |

**Termos técnicos traduzidos:**
| Técnico | Como dizemos |
|---------|-------------|
| LLM | modelo de linguagem |
| Prompt engineering | instruções do agente |
| RAG | agente com memória de documentos |
| Deploy | colocar em produção |
| Workflow | fluxo automatizado |

## Templates de Copy

### Site — Hero
```
Seu negócio operado por IA.
Squads de agentes inteligentes prontos para usar —
atendimento, vendas, financeiro e operação automatizados em dias, não meses.
[ Automatize sua empresa ]  [ Ver como funciona ]
```

### LinkedIn — Resultado de Cliente
```
[NÚMERO DE IMPACTO]
O que aconteceu: → [setor] → [squad] → [resultado em X dias]
O que mudou: → [métrica 1] → [métrica 2] → [tempo liberado]
Esse é o Squad [nome] da Eloscope.
Se seu [processo] ainda é manual, fale comigo.
```

### WhatsApp — Primeiro Contato
```
Olá [Nome], tudo bem?
Sou o Victor da Eloscope — estúdio de AI.
[Indicação] comentou que você está [contexto do problema].
Temos squads de agentes prontos para [área específica].
Em média, nossos clientes implementam em menos de 2 semanas.
Você tem 20 minutos essa semana para eu te mostrar como funciona?
```

### E-mail Frio
```
ASSUNTO: [Empresa] + agentes de AI em [X] dias
[Nome], vi que a [empresa] atua em [setor] — geralmente perde tempo com [processo].
Na Eloscope implementamos squads prontos para isso. Clientes em [setor] reduziram [métrica] em [resultado].
Vale 20 minutos para eu mostrar como funcionaria na [empresa]?
Victor — Co-founder, Eloscope
```
