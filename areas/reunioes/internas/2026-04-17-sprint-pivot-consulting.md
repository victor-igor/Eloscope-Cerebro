### 2. Diagnóstico crítico — Victor verbaliza o problema

> *"A gente ficou muito preso em ferramenta, acabou parte estratégica dos nichos em si, a gente não foi muito a fundo. Foi quase 95% de ferramenta até agora."*

- Pouco dado de antes/depois nos clientes (Fran, Maqlam, Campos venderam, mas a gente não tem métrica de quanto melhorou).
- Conhecimento de nicho ficou superficial; foco foi tecnologia, não resultado.
- Software house disfarçada: para entregar o que prometemos custou muito back+front (mínimo R$ 4k/mês de DEV iniciante na conta), ticket atual não sustenta.

### 3. Avaliação dos 3 caminhos

| Caminho | Avaliação |
|---------|-----------|
| **Educacional puro (curso/mentoria)** | Saturado. "Todo mundo quer ensinar, ninguém quer fazer." Demora a colocar dinheiro no bolso. |
| **Software house / SaaS** | É onde estamos preso. Multi-tenant exige time, segurança, suporte — não é a hora. |
| **Consultoria estratégica com IA (intermédio)** | ✅ Caminho escolhido. Entrar no negócio do cliente, analisar, entregar playbook, usar IA como meio para resultado. |

### 4. Definição do novo modelo — IA Consulting de Outcome

- **Core insight (Victor):** *"A gente não vai vender ferramenta. A gente vai mostrar pra ela como ela vai usar e aumentar o resultado dela. A IA vai ser o meio. O foco principal vai ser o resultado que ela vai gerar."*
- **Promessa modelo Alfa:** *"Até o final do ano, dobrar/triplicar o faturamento com o mesmo número de colaboradores — usando IA."*
- **Dois tipos de IA pra entregar:**
  - **Generativa**: ferramenta para o time da pessoa produzir mais (designer + IA = 2x output, vendedor + script + IA, etc.).
  - **Preditiva**: análise do negócio (taxa conversão, funil, dados), identificar gaps e oportunidades.
- **Stack preferida:** OpenCloud / Cloud Code + Skills + Hostinger (instalação rápida) — N8N sai do core, fica como complemento eventual.
- **Não vamos nichar segmento** (Eloscope não tem profundidade ainda) — vamos **nichar a solução / dor**.
- **Dor candidata #1:** comercial (todos os clientes atuais entraram com dor comercial: conversão, reativação, atendimento).

### 5. Como conectar IA + entrega

- Cliente pode usar ferramenta de mercado (Go High Level, etc.) — Eloscope **integra, configura e treina**, em vez de desenvolver tudo do zero.
- Quando precisar de IA própria, usa OpenCloud + Skills (rápido de montar — agente básico em 1-2 dias, com documentação pronta).
- Limitar agente ao escopo (ex.: Morgana sem envio de imagem) — IA faz triagem, humano fecha.
- Pra clientes existentes (Matheus, Fran, Denis): já tem onde aplicar essa nova abordagem **agora** — base própria pra testar antes de atacar leads novos.

### 6. Enertelles — caso prático aplicando o pivot

- Proposta concorrente vendia "infraestrutura" (250 minisites + Black Scale) — Lucas demonstrou via Comet/pesquisa real que o caminho contradiz SEO 2026 (Generative Engine Optimization > minisites).
- Nossa abordagem: site institucional + LP com cálculo de economia + blog automático para palavras-chave + SDR (R$ 4k–6k ticket).
- Cliente faz 100-200k/mês com energia solar mas não tem visão comercial — caso clássico do nosso novo target.
