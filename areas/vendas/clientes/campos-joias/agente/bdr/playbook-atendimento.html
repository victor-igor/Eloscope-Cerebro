P1 — Ainda trabalha com anéis de formatura?
→ Sim → continua para P2
→ Não → "Entendo! Se retomar, pode contar com a gente. Abraços!" → encerrar
P2 — Quantas turmas atende por temporada?
→ Qualquer resposta → anotar e ir para P3
P3 — Tem mostruário conosco?
→ Não tem → abrir venda de mostruário (Fase 4) antes de continuar
→ Tem → "Ótimo! O mesmo serve pra essa linha também." → P4
P4 — Já conhece a linha Grau 10K?
→ Não conhece → Fase 2 (Apresentação)
→ Conhece mas nunca vendeu → identificar objeção → tratar com scripts
→ Já vendeu → Fase 3 (Simulação) direto
P5 — Tem turma formando agora ou nos próximos meses?
→ Sim → priorizar simulação de custo
→ Não → foco no mostruário + catálogo para próxima temporada
